Answer Formatting

The root note should be capitalized, followed by a '#' if the chord is sharp, or a lower-case 'b' if the chord is flat. If the chord is minor, it should be followed by 'm'. Otherwise, no addition input is necessary. All whitespace, such as spaces, is ignored.
